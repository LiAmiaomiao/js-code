    - 杨辉三角
    
      - 题目描述
    
        ```
        输入: 5
        输出:
        [
             [1],
            [1,1],
           [1,2,1],
          [1,3,3,1],
         [1,4,6,4,1]
        ]
        ```
    
      - 解题思路
    
        - 判断如果不是该列数组的首位或者最后一位，则值为`a[i-1][j-1] + a[i-1][j]` ，否则值为`1` 
    
      - 实现代码
    
        ```
        function generate(numRows) {
            const result = [];
            if (numRows <= 0) {
                return result;
            }
            let i = 0, j = 0;
            for (let i = 0; i < numRows; i ++) {
                const subArr = [];
                for (let j = 0; j <= i; j++) {
                    if (j > 0 && j < i) {
                        subArr.push(result[i-1][j-1] + result[i-1][j]);
                    } else {
                        subArr.push(1);
                    }
                }
                result.push(subArr);
            }
            return result;
        };
        ```
